# E-Plant-Shopping Lab

##Introduction

###Paradise Nursery is a modern e‑commerce web application built with React and Redux, designed to provide a smooth and intuitive plant‑shopping experience. The project showcases core front‑end development concepts such as state management, component‑based architecture, and responsive UI design.
Users can browse a curated selection of plants, view detailed product information, manage their shopping cart, and explore a clean, user‑friendly interface inspired by real online plant stores. This project demonstrates practical skills in building scalable web applications and highlights the power of React and Redux in creating dynamic, interactive user experiences.

##Features

- **Plant Catalog:** Browse a curated list of plants with images, descriptions, and pricing.
- **Shopping Cart:** Add, remove, and update items with real-time state management via Redux.
- **Responsive UI:** Clean, mobile-friendly layout for a smooth shopping experience on any device.
- **State Management:** Centralized global state using Redux Toolkit for predictable and scalable logic.
